# 🚀 SEO Quick Start - Get Ranked in 24 Hours!

## ⚡ Do These 5 Things NOW!

### 1. **Google Search Console** (5 minutes)

```bash
1. Go to: https://search.google.com/search-console
2. Click "Add Property"
3. Enter your domain
4. Verify (HTML file or DNS)
5. Submit sitemap: https://yourdomain.com/sitemap.xml
```

### 2. **Bing Webmaster Tools** (3 minutes)

```bash
1. Go to: https://www.bing.com/webmasters
2. Import from Google Search Console (easiest!)
3. Or add manually
4. Submit sitemap
```

### 3. **IndexNow Setup** (2 minutes)

```bash
# Step 1: Generate key at https://www.indexnow.org/

# Step 2: Add to .env.local
INDEXNOW_KEY=your-key-here
SUBMIT_URL_SECRET=any-secret-key

# Step 3: Create file: public/[your-key].txt
# Content: your-key-here

# Step 4: Test
curl -X POST https://yourdomain.com/api/submit-url \
  -H "Content-Type: application/json" \
  -d '{"url":"https://yourdomain.com","secret":"your-secret"}'
```

### 4. **Submit Homepage** (1 minute)

```bash
# In Google Search Console:
1. Click "URL Inspection"
2. Enter: https://yourdomain.com
3. Click "Request Indexing"
```

### 5. **Ping Sitemap** (1 minute)

```bash
# Visit these URLs in browser:
https://www.google.com/ping?sitemap=https://yourdomain.com/sitemap.xml
https://www.bing.com/ping?sitemap=https://yourdomain.com/sitemap.xml
```
